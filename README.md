#  News Headlines Web Scraper

##  Objective:
To build a GUI-based Python application that scrapes the top news headlines from a public news website and saves them into a CSV file.

---

##  Tools & Technologies Used:
- Python
- requests
- BeautifulSoup
- tkinter (for GUI)
- pandas (for CSV file handling)

---

##  Features:
- Fetches live news headlines from a website.
- Displays the headlines in a scrollable listbox UI.
- Allows users to save headlines to a CSV file with one click.
- User-friendly GUI built with Tkinter.
- Error-handling for network failures or parsing issues.

---

##  Files in the Project:
- `news_scraper_gui.py` → Main Python script with GUI and scraping logic.
- `headlines.csv` → Auto-generated file after clicking 'Save to CSV'.

---

##  How It Works:
1. Click the “Fetch Headlines” button in the app.
2. Headlines are scraped from a news website and shown in the GUI.
3. Click “Save to CSV” to save the displayed headlines in `headlines.csv`.

---

##  How to Run:
1. Clone or download this repository.
2. Install required modules:
```

pip install requests beautifulsoup4 pandas

```
3. Run the script:
```

python webscarpper.py

```

---

## Outcome:
A beginner-friendly yet powerful desktop GUI app that automates news headline scraping and lets you save data for later analysis.


---

## Note:
- Ensure you are connected to the internet.
- The target news site structure must remain consistent for accurate scraping.
